# Blog Preview Card

This project is a responsive Blog Preview Card built as a Frontend Mentor challenge.  
It adapts well to mobile, tablet, and desktop screens.

## Overview
A clean, modern blog preview card that displays an image, category, date, title, author info, and a short description.  
The layout is designed to be responsive and visually appealing.

## Technologies
- CSS Flexbox
- Google Fonts (Figtree)
- Mobile-first methodology
- Hover effects for interactive elements

## Features
- Fully responsive design
- Hover effects on the title
- Clean typography with Figtree font
- Works on mobile and tablet screens

## Live Demo
Check out the live version here: [Live Site URL](https://treasure05nxazonke-art.github.io/blog-preview-card/)

## Challenges
- Adjusting image and card sizing for mobile and tablet screens.
- Solved using `max-width`, `object-fit`, and Flexbox alignment.

## Future Improvements
- Add more dynamic hover animations
- Optimize spacing for larger screens
